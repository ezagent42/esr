# ESR v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring the ESR v0.1 design (`docs/superpowers/specs/2026-04-18-esr-extraction-design.md`) to a working state where the 8-track E2E platform validation passes end-to-end, extracted from cc-openclaw's channel_server.

**Architecture:** Elixir/OTP runtime (PeerServer GenServers + Phoenix Channels for IPC) plus three Python layers (pure-function handlers, factory-style adapters, declarative command compiler). See spec §2.

**Tech Stack:**
- **Elixir**: OTP 27+, Phoenix 1.7+, libcluster (v0.2), ETS, :telemetry
- **Python**: 3.11+, `uv`, pydantic v2, aiohttp (Phoenix channel client via `phoenix_channels_client` or custom WS), `pytest`, `pytest-asyncio`
- **Tooling**: `mix` for Elixir, `uv` for Python, `ruff`/`mypy` for Python lint/type, `mix credo`/`mix dialyzer` for Elixir
- **Testing**: Elixir `ExUnit`, Python `pytest`, plus scenario runner in `scenarios/`

```elixir
# runtime/lib/esr/peer_registry.ex
defmodule Esr.PeerRegistry do
  @moduledoc """
  Registry of live PeerServers by actor_id. Thin wrapper over Elixir's
  Registry (keys: :unique). Used by PeerSupervisor for spawn/lookup and
  by message routing for destination resolution.
  """

  @registry __MODULE__

  @doc "Register the calling process (or a given pid) under actor_id."
  @spec register(String.t(), pid()) :: {:ok, pid()} | {:error, term()}
  def register(actor_id, pid) when is_binary(actor_id) and is_pid(pid) do
    case Registry.register(@registry, actor_id, pid) do
      {:ok, _} -> {:ok, pid}
      {:error, reason} -> {:error, reason}
    end
  end

  @doc "Look up the pid for an actor_id."
  @spec lookup(String.t()) :: {:ok, pid()} | :error
  def lookup(actor_id) do
    case Registry.lookup(@registry, actor_id) do
      [{pid, _value}] -> {:ok, pid}
      [] -> :error
    end
  end

  @doc "Enumerate all registered (actor_id, pid) pairs."
  @spec list_all() :: [{String.t(), pid()}]
  def list_all do
    Registry.select(@registry, [{{:"$1", :"$2", :_}, [], [{{:"$1", :"$2"}}]}])
  end
end
```

```elixir
# runtime/lib/esr/peer_server.ex
defmodule Esr.PeerServer do
  @moduledoc """
  One GenServer per live actor. Owns the actor's state; calls out to
  HandlerRouter on events, dispatches returned actions. This skeleton
  supports only state/get — event handling lands in Task 3.x after IPC
  is in place.
  """
  use GenServer

  require Logger

  defstruct [:actor_id, :actor_type, :handler_module, :state, :metadata]

  # ------------------------------------------------------------------
  # Public API
  # ------------------------------------------------------------------

  def start_link(opts) do
    actor_id = Keyword.fetch!(opts, :actor_id)
    GenServer.start_link(__MODULE__, opts, name: via(actor_id))
  end

  def get_state(actor_id) do
    GenServer.call(via(actor_id), :get_state)
  end

  defp via(actor_id) do
    {:via, Registry, {Esr.PeerRegistry, actor_id}}
  end

  # ------------------------------------------------------------------
  # GenServer callbacks
  # ------------------------------------------------------------------

  @impl true
  def init(opts) do
    state = %__MODULE__{
      actor_id: Keyword.fetch!(opts, :actor_id),
      actor_type: Keyword.fetch!(opts, :actor_type),
      handler_module: Keyword.fetch!(opts, :handler_module),
      state: Keyword.get(opts, :initial_state, %{}),
      metadata: Keyword.get(opts, :metadata, %{})
    }

    :telemetry.execute([:esr, :actor, :spawned], %{}, %{
      actor_id: state.actor_id,
      actor_type: state.actor_type
    })

    {:ok, state}
  end

  @impl true
  def handle_call(:get_state, _from, %__MODULE__{state: s} = acc) do
    {:reply, s, acc}
  end
end
```

```elixir
# runtime/lib/esr/uri.ex
defmodule Esr.Uri do
  @moduledoc """
  Parser / builder for esr:// URIs. Spec §7.5.

  Grammar:
      esr://[org@]host[:port]/<type>/<id>[?params]

  Host is REQUIRED — empty host is a syntax error. Types: actor, adapter,
  handler, command, interface (v0.1 supports actor, adapter, command).
  """

  @valid_types ~w(actor adapter handler command interface)a

  defstruct [:org, :host, :port, :type, :id, :params]

  @type t :: %__MODULE__{
          org: String.t() | nil,
          host: String.t(),
          port: non_neg_integer() | nil,
          type: atom(),
          id: String.t(),
          params: %{String.t() => String.t()}
        }

  @spec parse(String.t()) :: {:ok, t()} | {:error, atom()}
  def parse("esr://" <> rest) do
    with {:ok, authority, path_and_query} <- split_authority(rest),
         {:ok, org, host, port} <- parse_authority(authority),
         {:ok, type, id, params} <- parse_path_and_query(path_and_query) do
      {:ok,
       %__MODULE__{
         org: org,
         host: host,
         port: port,
         type: type,
         id: id,
         params: params
       }}
    end
  end

  def parse(_), do: {:error, :bad_scheme}

  @spec build(atom(), String.t(), String.t()) :: String.t()
  def build(type, id, host) when type in @valid_types do
    "esr://#{host}/#{type}/#{id}"
  end

  # ---- internal -----------------------------------------------------

  defp split_authority(rest) do
    case String.split(rest, "/", parts: 2) do
      [authority, path_and_query] -> {:ok, authority, "/" <> path_and_query}
      _ -> {:error, :no_path}
    end
  end

  defp parse_authority(""), do: {:error, :empty_host}

  defp parse_authority(authority) do
    {org, host_port} =
      case String.split(authority, "@", parts: 2) do
        [h] -> {nil, h}
        [o, h] -> {o, h}
      end

    case String.split(host_port, ":", parts: 2) do
      [""] -> {:error, :empty_host}
      [host] -> {:ok, org, host, nil}
      [host, port_str] ->
        case Integer.parse(port_str) do
          {port, ""} -> {:ok, org, host, port}
          _ -> {:error, :bad_port}
        end
    end
  end

  defp parse_path_and_query(path_and_query) do
    {path, params} = split_query(path_and_query)

    case String.split(path, "/", trim: true) do
      [type_str, id] ->
        type = String.to_atom(type_str)

        if type in @valid_types do
          {:ok, type, id, params}
        else
          {:error, :unknown_type}
        end

      _ ->
        {:error, :bad_path}
    end
  end

  defp split_query(path) do
    case String.split(path, "?", parts: 2) do
      [p] -> {p, %{}}
      [p, q] -> {p, URI.decode_query(q)}
    end
  end
end
```

```elixir
# runtime/lib/esr/telemetry/buffer.ex
defmodule Esr.Telemetry.Buffer do
  @moduledoc """
  ETS-backed rolling buffer of telemetry events. Retention window
  configurable (default 15 minutes via application config).
  """
  use GenServer

  @table :esr_telemetry_buffer

  defmodule Event do
    defstruct [:ts_unix_ms, :event, :measurements, :metadata]
  end

  # -- public --

  def start_link(opts) do
    GenServer.start_link(__MODULE__, opts, name: __MODULE__)
  end

  @spec record([atom()], map(), map()) :: :ok
  def record(event, measurements \\ %{}, metadata \\ %{}) do
    now = System.os_time(:millisecond)

    :ets.insert(@table, {
      {now, make_ref()},
      %Event{
        ts_unix_ms: now,
        event: event,
        measurements: measurements,
        metadata: metadata
      }
    })

    :ok
  end

  @spec query(keyword()) :: [Event.t()]
  def query(opts) do
    duration_s = Keyword.get(opts, :duration_seconds, 900)
    floor = System.os_time(:millisecond) - duration_s * 1_000

    :ets.select(@table, [
      {
        {{:"$1", :_}, :"$2"},
        [{:>=, :"$1", floor}],
        [:"$2"]
      }
    ])
  end

  # -- GenServer --

  @impl true
  def init(opts) do
    retention_min = Keyword.get(opts, :retention_minutes, 15)
    :ets.new(@table, [:named_table, :ordered_set, :public, read_concurrency: true])
    schedule_prune(retention_min)
    {:ok, %{retention_min: retention_min}}
  end

  @impl true
  def handle_info(:prune, %{retention_min: mins} = state) do
    floor = System.os_time(:millisecond) - mins * 60 * 1_000
    :ets.select_delete(@table, [{{{:"$1", :_}, :_}, [{:<, :"$1", floor}], [true]}])
    schedule_prune(mins)
    {:noreply, state}
  end

  defp schedule_prune(mins), do: Process.send_after(self(), :prune, mins * 60 * 1_000)
end
```

```python
# py/src/esr/handler.py
"""Handler decorator. Spec §4.

@handler(actor_type, name) registers a pure function
  (state, event) -> (new_state, [Action])

@handler_state(actor_type, schema_version) registers the state model.

Purity enforcement lives in esr.verify.purity (CI-time).
"""
from __future__ import annotations

from dataclasses import dataclass
from typing import Any, Callable


@dataclass(frozen=True)
class HandlerEntry:
    actor_type: str
    name: str
    fn: Callable


@dataclass(frozen=True)
class StateEntry:
    actor_type: str
    schema_version: int
    model: type


HANDLER_REGISTRY: dict[str, HandlerEntry] = {}
STATE_REGISTRY: dict[str, StateEntry] = {}


def handler(*, actor_type: str, name: str):
    def decorator(fn: Callable) -> Callable:
        key = f"{actor_type}.{name}"
        if key in HANDLER_REGISTRY:
            raise ValueError(f"handler {key} already registered")
        HANDLER_REGISTRY[key] = HandlerEntry(actor_type, name, fn)
        return fn
    return decorator


def handler_state(*, actor_type: str, schema_version: int = 1):
    def decorator(cls: type) -> type:
        if actor_type in STATE_REGISTRY:
            raise ValueError(f"state for actor_type={actor_type} already registered")
        STATE_REGISTRY[actor_type] = StateEntry(actor_type, schema_version, cls)
        return cls
    return decorator
```

```python
# py/src/esr/adapter.py
"""Adapter decorator + config helper. Spec §5."""
from __future__ import annotations

from dataclasses import dataclass
from typing import Any


@dataclass(frozen=True)
class AdapterEntry:
    name: str
    cls: type
    allowed_io: dict[str, Any]


ADAPTER_REGISTRY: dict[str, AdapterEntry] = {}


class AdapterConfig:
    """Read-only wrapper over the config dict; attribute-style access."""

    def __init__(self, data: dict):
        object.__setattr__(self, "_data", dict(data))

    def __getattr__(self, name: str) -> Any:
        try:
            return self._data[name]
        except KeyError as err:
            raise AttributeError(name) from err

    def __setattr__(self, *_):
        raise AttributeError("AdapterConfig is read-only")


def adapter(*, name: str, allowed_io: dict[str, Any]):
    def decorator(cls: type) -> type:
        if not hasattr(cls, "factory") or not callable(getattr(cls, "factory", None)):
            raise TypeError(f"adapter class must define a static factory method")
        if name in ADAPTER_REGISTRY:
            raise ValueError(f"adapter {name} already registered")
        ADAPTER_REGISTRY[name] = AdapterEntry(name, cls, dict(allowed_io))
        return cls
    return decorator
```

```python
# py/src/esr/command.py
"""Command EDSL + compiler. Spec §6."""
from __future__ import annotations

import contextlib
import contextvars
from dataclasses import dataclass, field
from typing import Any, Callable

# -- types ---------------------------------------------------------------

@dataclass
class _Node:
    id: str
    actor_type: str
    handler: str
    adapter: str | None = None
    params: dict = field(default_factory=dict)
    _depends_on: list = field(default_factory=list)

    def __rshift__(self, other: "_Node") -> "_Node":
        # a >> b records an edge from a to b in the current command context
        _record_edge(self, other)
        return other


@dataclass
class _Port:
    name: str
    type: str
    direction: str  # "input" | "output"


@dataclass(frozen=True)
class Edge:
    from_id: str
    to_id: str


@dataclass(frozen=True)
class Topology:
    name: str
    nodes: tuple
    edges: tuple
    ports_in: tuple
    ports_out: tuple
    params: tuple  # parameter definitions, derived from {{var}} templates


@dataclass
class CommandEntry:
    name: str
    fn: Callable


COMMAND_REGISTRY: dict[str, CommandEntry] = {}


# -- context for building a command --------------------------------------

@dataclass
class _CommandContext:
    name: str
    nodes: list[_Node] = field(default_factory=list)
    edges: list[Edge] = field(default_factory=list)
    ports_in: list[_Port] = field(default_factory=list)
    ports_out: list[_Port] = field(default_factory=list)


_ctx: contextvars.ContextVar[_CommandContext | None] = contextvars.ContextVar(
    "_esr_cmd_ctx", default=None
)


@contextlib.contextmanager
def _binding(name: str):
    tok = _ctx.set(_CommandContext(name=name))
    try:
        yield _ctx.get()
    finally:
        _ctx.reset(tok)


def _record_node(n: _Node) -> None:
    ctx = _ctx.get()
    if ctx is None:
        raise RuntimeError("node() called outside @command function")
    ctx.nodes.append(n)


def _record_edge(src: _Node, dst: _Node) -> None:
    ctx = _ctx.get()
    if ctx is None:
        raise RuntimeError("edge recorded outside @command function")
    ctx.edges.append(Edge(from_id=src.id, to_id=dst.id))


# -- public EDSL ---------------------------------------------------------

def node(
    *,
    id: str,
    actor_type: str,
    handler: str,
    adapter: str | None = None,
    params: dict | None = None,
    depends_on: list[_Node] | None = None,
) -> _Node:
    n = _Node(
        id=id,
        actor_type=actor_type,
        handler=handler,
        adapter=adapter,
        params=dict(params or {}),
        _depends_on=list(depends_on or []),
    )
    _record_node(n)
    return n


class port:
    @staticmethod
    def input(name: str, type: str) -> str:
        ctx = _ctx.get()
        if ctx is None:
            raise RuntimeError("port.input called outside @command function")
        ctx.ports_in.append(_Port(name=name, type=type, direction="input"))
        return name  # the id template uses the port name

    @staticmethod
    def output(name: str, type: str) -> str:
        ctx = _ctx.get()
        if ctx is None:
            raise RuntimeError("port.output called outside @command function")
        ctx.ports_out.append(_Port(name=name, type=type, direction="output"))
        return name


def command(name: str):
    def decorator(fn: Callable) -> Callable:
        if name in COMMAND_REGISTRY:
            raise ValueError(f"command {name} already registered")
        COMMAND_REGISTRY[name] = CommandEntry(name=name, fn=fn)
        return fn
    return decorator


# -- compiler ------------------------------------------------------------

import re


def _extract_params(nodes: list[_Node]) -> tuple:
    pat = re.compile(r"\{\{\s*([a-zA-Z_][\w.]*)\s*\}\}")
    found = set()
    for n in nodes:
        for haystack in (n.id, n.actor_type, n.handler, n.adapter or "", *map(str, n.params.values())):
            found.update(pat.findall(haystack))
    return tuple(sorted(found))


def _check_dag(nodes: list[_Node]) -> None:
    # Kahn's algorithm on depends_on
    ids = {n.id: n for n in nodes}
    indeg = {nid: 0 for nid in ids}
    for n in nodes:
        for dep in n._depends_on:
            indeg[n.id] += 1
    ready = [nid for nid, d in indeg.items() if d == 0]
    visited = []
    while ready:
        nid = ready.pop()
        visited.append(nid)
        for n in nodes:
            if ids.get(n.id) is ids.get(nid):
                continue
            if any(dep.id == nid for dep in n._depends_on):
                indeg[n.id] -= 1
                if indeg[n.id] == 0:
                    ready.append(n.id)
    if len(visited) < len(nodes):
        raise ValueError("depends_on cycle detected")


def compile_topology(name: str) -> Topology:
    entry = COMMAND_REGISTRY[name]
    with _binding(name) as ctx:
        entry.fn()

    _check_dag(ctx.nodes)
    params = _extract_params(ctx.nodes)

    return Topology(
        name=name,
        nodes=tuple(ctx.nodes),
        edges=tuple(ctx.edges),
        ports_in=tuple(ctx.ports_in),
        ports_out=tuple(ctx.ports_out),
        params=params,
    )
```

```python
# py/src/esr/uri.py
"""esr:// URI parser and builder. Spec §7.5."""
from __future__ import annotations

from dataclasses import dataclass
from urllib.parse import urlsplit, parse_qs

VALID_TYPES = {"actor", "adapter", "handler", "command", "interface"}


@dataclass(frozen=True)
class EsrURI:
    host: str
    port: int | None
    type: str
    id: str
    org: str | None = None
    params: dict | None = None


def parse(uri: str) -> EsrURI:
    if not uri.startswith("esr://"):
        raise ValueError("URI must start with esr://")

    s = urlsplit(uri)
    if not s.hostname:
        raise ValueError("empty host — esr:// URIs require an explicit host")

    path_parts = [p for p in s.path.split("/") if p]
    if len(path_parts) != 2:
        raise ValueError(f"expected /<type>/<id>, got {s.path!r}")

    type_, id_ = path_parts
    if type_ not in VALID_TYPES:
        raise ValueError(f"unknown type: {type_}")

    org = s.username
    params = {k: v[0] for k, v in parse_qs(s.query).items()} if s.query else None

    return EsrURI(
        host=s.hostname,
        port=s.port,
        type=type_,
        id=id_,
        org=org,
        params=params,
    )


def build(type_: str, id_: str, *, host: str, port: int | None = None, org: str | None = None) -> str:
    if not host:
        raise ValueError("host is required")
    authority = host
    if port is not None:
        authority = f"{host}:{port}"
    if org:
        authority = f"{org}@{authority}"
    return f"esr://{authority}/{type_}/{id_}"
```

```python
# py/src/esr/ipc/channel_client.py
"""Lightweight Phoenix-channel client over aiohttp WebSocket.

Phoenix channel messages are arrays: [join_ref, ref, topic, event, payload].
We multiplex many topics over one socket.
"""
from __future__ import annotations

import asyncio
import json
import logging
from typing import Awaitable, Callable

import aiohttp

log = logging.getLogger(__name__)

PhxFrame = list


class ChannelClient:
    """Connects to a Phoenix channels endpoint and joins a set of topics."""

    def __init__(self, url: str):
        self.url = url
        self._ws: aiohttp.ClientWebSocketResponse | None = None
        self._session: aiohttp.ClientSession | None = None
        self._ref = 0
        self._handlers: dict[str, Callable[[dict], Awaitable[None]]] = {}
        self._recv_task: asyncio.Task | None = None

    async def connect(self) -> None:
        self._session = aiohttp.ClientSession()
        self._ws = await self._session.ws_connect(self.url)
        self._recv_task = asyncio.create_task(self._recv_loop())

    async def join(self, topic: str, on_msg: Callable[[dict], Awaitable[None]]) -> None:
        self._handlers[topic] = on_msg
        await self._send_frame([None, self._next_ref(), topic, "phx_join", {}])

    async def push(self, topic: str, event: str, payload: dict) -> None:
        await self._send_frame([None, self._next_ref(), topic, event, payload])

    async def close(self) -> None:
        if self._recv_task:
            self._recv_task.cancel()
        if self._ws:
            await self._ws.close()
        if self._session:
            await self._session.close()

    # -- internals --

    def _next_ref(self) -> str:
        self._ref += 1
        return str(self._ref)

    async def _send_frame(self, frame: PhxFrame) -> None:
        assert self._ws is not None
        await self._ws.send_str(json.dumps(frame))

    async def _recv_loop(self) -> None:
        assert self._ws is not None
        async for msg in self._ws:
            if msg.type != aiohttp.WSMsgType.TEXT:
                continue
            try:
                frame = json.loads(msg.data)
            except json.JSONDecodeError:
                log.warning("unparseable frame: %r", msg.data)
                continue
            if len(frame) < 5:
                continue
            _, _, topic, _event, payload = frame
            handler = self._handlers.get(topic)
            if handler is None:
                continue
            try:
                await handler(payload)
            except Exception:
                log.exception("handler for %s raised", topic)
```

```python
"""A long-running Python worker that joins handler:<module>/<worker_id>,
receives handler_call envelopes, invokes the registered handler, and
returns handler_reply. Spawned by the Elixir HandlerRouter.Pool."""
from __future__ import annotations

import asyncio
import logging
import sys
import uuid
from dataclasses import asdict

from esr.handler import HANDLER_REGISTRY, STATE_REGISTRY
from esr.actions import Emit, Route, InvokeCommand
from esr.events import Event
from esr.ipc.channel_client import ChannelClient
from esr.ipc.envelope import make_handler_reply, dumps

log = logging.getLogger(__name__)


def _serialise_action(a):
    base = asdict(a)
    if isinstance(a, Emit): base["type"] = "emit"
    elif isinstance(a, Route): base["type"] = "route"
    elif isinstance(a, InvokeCommand): base["type"] = "invoke_command"
    return base


async def run(url: str, module: str, worker_id: str | None = None):
    worker_id = worker_id or uuid.uuid4().hex[:8]
    client = ChannelClient(url)
    await client.connect()

    topic = f"handler:{module}/{worker_id}"
    source = f"esr://localhost/handler/{module}"

    async def on_msg(payload):
        if payload.get("type") != "handler_call":
            return
        call_id = payload["id"]
        inner = payload["payload"]
        state_dict = inner["state"]
        event_dict = inner["event"]

        entry = HANDLER_REGISTRY.get(inner["handler"])
        if entry is None:
            log.error("handler not registered: %s", inner["handler"])
            return

        state_entry = STATE_REGISTRY.get(entry.actor_type)
        state = state_entry.model(**state_dict) if state_entry else state_dict
        event = Event(**event_dict)

        new_state, actions = entry.fn(state, event)

        await client.push(topic, "handler_reply", {
            "id": call_id,
            "type": "handler_reply",
            "source": source,
            "payload": {
                "new_state": new_state.model_dump() if state_entry else new_state,
                "actions": [_serialise_action(a) for a in actions],
            },
        })

    await client.join(topic, on_msg)
    # run forever
    await asyncio.Event().wait()


if __name__ == "__main__":
    url = sys.argv[1]
    module = sys.argv[2]
    asyncio.run(run(url, module))
```

```python
# handlers/feishu_app/src/feishu_app/on_msg.py
from __future__ import annotations

from esr.handler import handler
from esr.events import Event
from esr.actions import Emit, InvokeCommand, Route
from .state import FeishuAppState

_NEW_THREAD_PREFIX = "/new-thread "


@handler(actor_type="feishu_app_proxy", name="on_msg")
def on_msg(state: FeishuAppState, event: Event) -> tuple[FeishuAppState, list]:
    if event.event_type != "msg_received":
        return state, []

    content = event.args.get("content", "")

    if content.startswith(_NEW_THREAD_PREFIX):
        thread_name = content[len(_NEW_THREAD_PREFIX):].strip()
        if not thread_name:
            return state, []  # malformed; ignore
        if thread_name in state.bound_threads:
            return state, []  # idempotent
        return (
            state.with_thread(thread_name),
            [InvokeCommand("feishu-thread-session", {"thread_id": thread_name})],
        )

    thread_id = event.args.get("thread_id")
    if thread_id and thread_id in state.bound_threads:
        return state, [Route(target=f"thread:{thread_id}", msg=content)]

    return state, []
```

## Post-implementation: Full Subagent Code Review

After all phases complete, dispatch a **full-scope** `superpowers:code-reviewer` subagent (not Elixir-only). The review covers:

- Python SDK + CLI (`py/`, `handlers/`, `adapters/`, `patterns/`) — architecture, API ergonomics, test coverage, purity enforcement, security
- Elixir runtime (`runtime/`) — OTP supervision strategies, GenServer state discipline, anti-patterns, Phoenix channel handling, telemetry correctness
- IPC layer — envelope consistency between Python and Elixir, timeout/retry/error handling, serialisation correctness
- E2E scenario — does it actually exercise the acceptance criteria in spec §9?
- Docs — are `docs/design/` v0.3 reference notes consistent with what got built?

**Elixir-specific checks** are performed by the same reviewer using the `.claude/skills/elixir-phoenix-helper/` references:

- `references/otp-core.md` — supervision, GenServer state, let-it-crash discipline
- `references/phoenix-web.md` — channel lifecycle, endpoint config, LiveView (n/a here) idioms
- `references/anti-patterns.md` — misuse of GenServer state, scattered process interfaces, boolean obsession, deprecated APIs (Logger.warn, use Mix.Config, etc.)
- `references/testing.md` — ExUnit patterns, async vs sync tests, property-based opportunities
- `references/realtime.md` — Phoenix PubSub + Channels patterns
- `references/macros.md`, `references/ecto.md` — applicable if we introduce either (Ecto not in v0.1)

The reviewer reads these alongside the Elixir source and flags deviations. Python gets the equivalent treatment via ruff/mypy plus reviewer judgement — no dedicated Python skill is required because the Python conventions are simpler and the reviewer can apply general Python best practice.

**Review output:** report structured as `## Critical / ## Significant / ## Minor / ## Strengths` with section/line references. Any critical findings halt merge; significant findings file issues; minor findings get bulk-fixed.

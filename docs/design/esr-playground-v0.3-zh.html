这是一个 crossbar switch matrix: 每个格子 (src, tar) 代表一条潜在的消息路径。有小点的格子是 contract 允许的路由, 没小点的是禁止的。消息触发时相应格子点亮 (颜色=topic), 违反契约时红光 + ✗。点击任意点亮过的格子可以看该路由上流过的所有消息; 点击左侧 agent 行头可以手动发消息。
💡 点击 agent 行头 手动发消息 · 点击格子查看该路由的消息历史 · 对角线/暗格子表示不可达

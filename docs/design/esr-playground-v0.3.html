switch matrix · rows = sender, cols = receiver · each lit cell is one routed message · click to inspect

This is a crossbar switch matrix: each cell (src, tar) represents one possible message route. Cells with a dot are contract-permitted; cells without are denied. When a message fires, the corresponding cell lights up (color = topic); violations show red glow + ✗. Click any cell that has lit up to see every message that flowed through it; click a row header on the left to send manually.
💡 Click row header to send · click a cell for its message history · diagonal/dim cells = unreachable
